# Experiments
The repetitions are done using copilot in Doocker container.
The access token is shared using Copilot API (you should install copilot in your system first, authenticate and try a simple command like: 

copilot --model "gpt-5-mini" --silent --no-custom-instructions --available-tools "" -p "hello world"



For the experiment:
* all the external connections are disabled
* an empty context (`copilot_emptyt_context` is created)

Guide:
https://docs.github.com/en/copilot/how-tos/copilot-cli/use-copilot-cli
